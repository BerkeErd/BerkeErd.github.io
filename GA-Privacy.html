Privacy Policy for Game Analytics

Effective Date: 03.25.2025

Introduction

("we", "our", "us") uses GameAnalytics to collect and analyze data for the purpose of improving our game's performance, understanding user engagement, and enhancing the player experience.

Data We Collect

Through GameAnalytics, we may collect the following data:

  • Device Information: Device model, operating system, language, and region.
  • User Interaction Data: Game progress, in-game events, and usage patterns.
  • Advertising Identifiers: IDFA (for iOS devices) and GAID (for Android devices) for personalized advertising.
  • IP Addresses: Anonymized and not used to identify individuals.

How We Use the Data

The data collected by GameAnalytics helps us:

  • Analyze game performance and stability.
  • Improve user experience and game features.
  • Understand player preferences for future game updates.
  • Optimize in-game advertising.

Data Sharing

We do not share personal data with third parties except for analytics purposes through GameAnalytics. The data collected by GameAnalytics is processed according to their privacy policy, which can be reviewed here.

Security

We take reasonable measures to protect your data from unauthorized access, alteration, or destruction. GameAnalytics also adheres to industry-standard security practices to ensure data protection.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated effective date.
