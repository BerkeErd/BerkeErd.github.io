About The Game

An early online multiplayer prototype built with Unity WebGL, NativeWebSocket, Node.js, and Express. Players could create rooms, join through room codes, and synchronize match state through a lightweight backend. This early networking experiment later informed the experimental online co-op beta in Stick Slasher.

The original backend was hosted on Glitch and is no longer available, so the playable online version has been retired.
