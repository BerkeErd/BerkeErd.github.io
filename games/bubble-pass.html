About The Game

Bubble Pass is a unique game built for Mobile platforms using Unity Engine, released on Jun 14, 2022. It is a unique game where you try to change the color of the bottom of the cup according to the color of the falling balls, allowing the colors to pass through. The game features a leaderboard system implemented using PlayFab SDK.
